# PredictFun — этапы внедрения

## Этап 0 — Текущая база
Уже есть:
- News Monitor работает
- логика сигналов по MetaMask / OpenSea / Base действует
- hourly alerting настроен

## Этап 1 — Public market watcher
Что внедрять первым:
- чтение public market data
- матчинг 3 нужных рынков
- различение дат внутри каждого рынка
- baseline state
- сравнение предыдущего и текущего состояния
- детект:
  - движение вероятности
  - рост объёма
  - сдвиг last sale
  - аномалии orderbook
  - активизация отдельных дат

Что даёт:
- рыночный сигнал без private доступа
- безопасный следующий шаг

## Этап 2 — Private read-only
Только после подтверждения auth flow:
- orders
- positions
- balance
- account context

Что даёт:
- понимание, как сигнал влияет на текущие позиции пользователя

## Этап 3 — Recommendation layer
Не действия, а рекомендации:
- что посмотреть
- какой рынок ожил
- какие даты сместились
- где позиции могут быть под риском

## Этап 4 — Semi-auto обсуждается отдельно
Только при отдельном решении:
- order cancellation
- replace order
- other trading actions

Без явного подтверждения пользователя не включать.

## Что сейчас не внедрять
- auto trading
- auto reposition
- auto cancel/replace
- любые действия с ордерами без отдельного контура безопасности
